Array's vs Hashes

An array is a indexed collection of elements. The elements can be all of any type of object, strings, numbers but could be other type of objects also. Arrays are indexed by sequential numbers only, enabling you to refer to the element by its index number. Imagine series of numbered boxes, each box holding word. If you wanted to know what the word in box number 3 is you would, in english say 'tell me what is in box number 3', simple enough. You can refer to any of the boxes by their box number.

A hash, is also a collection of elements. The elements can also be any type of object like an array. The elements in a hash though consist of the object to be stored as well as a key element. Hashes store things in pairs, the pair consisting of a key and the object.

Both array's and hashes are structures for storing set's of objects that can referenced by their key

The primary difference with array's and hashes is that with a hash you can choose yourself what you want the indexed key to be, it could be anything but the best for it, in my view would be to have the key identify the type of object the key relates to, for example a hash pair might , that being the first element of the pair, whereas an with an array the index number is assigned to the object to be stored when it is added the array.

Some articles state it is more efficient accessing array elements, but hashes are more flexible due to the ability to assign the key of the object yourself. But I have found other articles that state hashes are more efficient thn array's. May have to get clarification on this point.

Some example's of the use of arrays and hashes

creating array named my_array and populating it with "dog","cat","rabbit","bird","fish". The order of the elements in the array are in the same order they appear between the brackets. So "dog" is the first array element, "cat" the second and so on.


my_array = ["dog","cat","rabbit","bird","fish"]

To access the array elements you just use an integer that refers to the elements order when the array was created or defined.


examples of how to access array elements my_array[2] would give you "cat" (accessing the 2nd array element) my_array[4] would give you "bird" (accessing the 4th array element)

To set or change array elements use the following syntax

my_array[1] = "rat" (sets the first array element to "rat", this will overwrite "dog" in this case my_array[3] = "pig" (sets the third array element to "pig", this will overwrite "rabbit")

Arrays and hashes are structures that enable in-memory storage of elements for further processing using the powerful methods for element and object manipulation like indexing, searching, replacing, stripping apart , and joining together to achieve the desired result
