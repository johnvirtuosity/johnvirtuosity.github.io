Looping in Ruby vs. Looping in JavaScript and Ruby hash vs. JavaScript Objects

JavaScript implements 2 looping constructs. These are 'for' and 'while' loops, there is also a 3rd which is variant of the while loop and that is the 'do..while'
Ruby implements 3. These include for , while, and until.

Ruby, additionally has a group of looping methods that enable you to loop through each of the elements in a collection. The collection is a group of enumerable elements.
These iterators include the 'each' iterator.
The syntax for the use of the iterator is ...
collection.each do |variable| code end In JavaScript, there is no 'each' iterator but with the use of the 'for' you can iterate over a group of elements much like you would if you used the .each iterator method in Ruby.

The functionality of looping constructs in Ruby, 'for ', While', and 'Until', can all be implemented in JavaScript by using JavaScripts for, while and Do..While loops

In JavaScript the for loop has the general syntax... for (var list_pointer ; condition ; increment_value){ code block }; The 'var list_pointer', and 'increment_value' are actually optional and can be set elsewhere. this feature makes the 'for' loop in JavaScript very flexible and configurable. Below is an example of this feature myPointer = 0;
for (; myPointer <= 20 ;){
console.log('Pointer is ' + myPointer)
myPointer += 2 //increment by 2
};
equivalent Ruby code using the 'step' iterator
[0..20].step do |myPointer|
puts "Pointer is #{myPointer}"
end

Ruby Hash vs. JavaScript Object

Hash objects are useful when you want to store and manipulate pairs or values that are related or associated with each other.

Ruby has a specific hash object that allows you to store key:value pairs where the key can be non-numeric.

JavaScript, doe's not include a specific 'hash' object, only an array object that requires a numbered key, but using JavaScript objects, you can create an object that behaves like a Ruby hash.



A comparison of how these look from a code perspective is below... Creating a new hash
'in Ruby'
myHash = { one => 'Joe Bloggs', two => 'Tim Smith', three => 'Amy Jones'}
Accessing a 'hash' value
myHash['one'] , returns 'Joe Bloggs'
Setting a hash value
myHash['three'] = 'Adam Black'
Adding a key:value pair
myHash['four'] = 'Terry White'
Deleting a key:value pair
myHash['three'].delete //deletes the key 'three' along with its value


'in JavaScript'
var myHash = {one: 'Joe Bloggs', two: 'Tim Smith', three: 'Amy Jones'};
Accessing a 'hash' value
myHash['one']; , returns 'Joe Bloggs'
Setting a hash value
myHash['three'] = 'Adam Black';
Adding a key:value pair
myHash['four'] = 'Terry White'; //Adds the 'key' 'four' and sets it's value to 'Terry White'
Deleting a key:value pair
delete myHash['three']; //deletes the key 'three' along with its value
